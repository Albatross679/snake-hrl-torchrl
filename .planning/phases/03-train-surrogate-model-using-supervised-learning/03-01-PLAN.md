   ```python

For the rollout loss block (lines 368-392): wrap it in an additional guard that checks `config.rollout_loss_weight > 0` AND prints a warning if TrajectoryDataset fails to load (since Phase 02.2 has no trajectory data). The simplest fix: add a try/except around TrajectoryDataset construction that catches the error and sets `config.rollout_loss_weight = 0` with a printed warning.
