  <name>Task 1: Create analysis script and generate diagnostic plots</name>
  <files>aprx_model_elastica/analyze_sweep.py, figures/surrogate_training/error_histograms.png, figures/surrogate_training/predicted_vs_actual.png, figures/surrogate_training/sweep_comparison.png, figures/surrogate_training/component_rmse.png</files>
  <action>
Create `aprx_model_elastica/analyze_sweep.py` that:

1. **Loads sweep results**: Read `output/surrogate/sweep_summary.json` to get all run names and val_loss values. For each run, load `metrics.json`.

2. **Sweep comparison chart** (`figures/surrogate_training/sweep_comparison.png`):
   - Horizontal bar chart of val_loss for all configs, sorted best to worst
   - Color-code: green for best, blue for others, red for failed
   - Include val_loss value as text label on each bar

3. **Per-component RMSE in physical units** (`figures/surrogate_training/component_rmse.png`):
   - For the best run: load model.pt, normalizer.pt, config.json
   - Run inference on full validation set (FlatStepDataset split="val")
   - Compute per-component RMSE in PHYSICAL UNITS:
     * pos_x, pos_y: RMSE in mm (multiply by 1000)
     * vel_x, vel_y: RMSE in mm/s (multiply by 1000)
     * yaw: RMSE in rad
     * omega_z: RMSE in rad/s
   - Also compute R-squared per component
   - Bar chart with RMSE values labeled, grouped by component type

4. **Error histograms** (`figures/surrogate_training/error_histograms.png`):
   - 6-panel figure (2x3): one histogram per component (pos_x, pos_y, vel_x, vel_y, yaw, omega_z)
   - Each shows distribution of prediction errors (pred - actual) in physical units
   - Mark mean and std on each panel

5. **Predicted vs actual** (`figures/surrogate_training/predicted_vs_actual.png`):
   - 6-panel scatter plot (2x3): predicted delta vs actual delta for each component
   - Subsample to ~5000 points for readability
   - Include diagonal line (perfect prediction) and R-squared annotation

6. **Print summary table** to stdout: run name, val_loss, rank. For best run: per-component RMSE and R-squared.

Use matplotlib with Agg backend (no display), dpi=150, bbox_inches="tight" per project convention.

CLI: `python3 -m aprx_model_elastica.analyze_sweep --output-base output/surrogate --data-dir data/surrogate_rl_step --figures-dir figures/surrogate_training`
  </action>
  <verify>
    <automated>python3 -m aprx_model_elastica.analyze_sweep --output-base output/surrogate --data-dir data/surrogate_rl_step --figures-dir figures/surrogate_training && ls figures/surrogate_training/*.png | wc -l</automated>
  </verify>
  <done>4 diagnostic plots saved to figures/surrogate_training/. Per-component RMSE in physical units printed. Sweep comparison chart shows all runs ranked.</done>
