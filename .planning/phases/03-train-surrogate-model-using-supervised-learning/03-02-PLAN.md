  <files>aprx_model_elastica/analyze_surrogate.py, figures/surrogate_training/*.png, output/surrogate/best/</files>
  <action>
Create `aprx_model_elastica/analyze_surrogate.py` with the following functionality:

**1. Sweep result loading and best model selection:**
- Scan `output/surrogate/sweep_*/metrics.json` to find all completed runs
- Parse best_val_loss from each, select the run with lowest val loss
- Print a ranked table of all runs with their val loss, LR, and hidden dims (from config.json)

**2. Load best model and evaluate on validation set:**
- Load SurrogateModel from best run's model.pt and config.json
- Load StateNormalizer from best run's normalizer.pt
- Load FlatStepDataset(split="val") from data/surrogate_rl_step_rl_step
- Run inference on entire val set in batches (batch_size=4096)
- For each sample: compute predicted delta (denormalized), compare to actual delta
- Compute per-component RMSE in raw units, then convert:
  - pos_x, pos_y: meters -> mm (multiply by 1000)
  - vel_x, vel_y: m/s -> mm/s (multiply by 1000)
  - yaw: radians (keep as-is)
  - omega_z: rad/s (keep as-is)
- Also compute per-component MAE and max absolute error
- Print a formatted table: component | RMSE (physical) | MAE (physical) | Max Error (physical)

**3. Generate plots (saved to figures/surrogate_training/):**

Plot 1 - `error_histograms.png`: 2x3 grid of histograms showing error distribution for each component (pos_x, pos_y, vel_x, vel_y, yaw, omega_z). X-axis in physical units. Use 100 bins, log-scale y-axis for visibility. Title each subplot with component name and RMSE value.

Plot 2 - `predicted_vs_actual.png`: 2x2 grid of scatter plots (predicted delta vs actual delta) for 4 key components: CoM_x delta (mean of pos_x deltas), CoM_y delta, mean velocity magnitude delta, mean heading delta. Include diagonal reference line. Use alpha=0.1 for density visibility. Show R^2 in corner.

Plot 3 - `sweep_comparison.png`: Horizontal bar chart of validation loss for each sweep config. Bars colored by hidden_dims (blue for 256x3, orange for 512x3). Annotate with actual val loss value. Highlight the best config.

Follow existing plot pattern from validate.py: matplotlib Agg backend, dpi=150, bbox_inches="tight".

**4. Copy best model to output/surrogate/best/:**
- `shutil.copytree(best_run_dir, "output/surrogate/best")` (or copy individual files)
- This is the canonical checkpoint location Phase 4 will consume

**5. CLI:**
- argparse with `--sweep-dir` (default "output/surrogate"), `--data-dir` (default "data/surrogate_rl_step"), `--figures-dir` (default "figures/surrogate_training"), `--device` (default "auto")
- Entry via `python3 -m aprx_model_elastica.analyze_surrogate` or `python3 aprx_model_elastica/analyze_surrogate.py`
