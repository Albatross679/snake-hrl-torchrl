<task type="auto">
  <name>Task 1: Build analysis script and generate diagnostic plots</name>
  <files>aprx_model_elastica/analyze_sweep.py, figures/surrogate_training/</files>
  <action>
Create aprx_model_elastica/analyze_sweep.py that:

1. Loads output/surrogate/sweep_summary.json to get all 15 config names and val_loss values.

2. For each config, loads model.pt + normalizer.pt + config.json from output/surrogate/{config_name}/.

3. Loads a sample of validation data from data/surrogate_rl_step/ via FlatStepDataset(split="val").

4. For the top 5 models by val_loss, runs inference on validation set and computes:
   - Per-component RMSE in PHYSICAL units:
     - pos_x, pos_y: multiply by 1000 to get mm (rod length ~0.2m scale)
     - vel_x, vel_y: multiply by 1000 to get mm/s
     - yaw: in radians (already physical)
     - omega_z: in rad/s (already physical)
   - Per-component MAE in same units
   - R-squared per component

5. Generates 4 diagnostic plots saved to figures/surrogate_training/:

   a. sweep_comparison.png: Horizontal bar chart of val_loss for all 15 configs, color-coded by architecture family (MLP=blue, Residual=green, Wide/Deep=orange, Transformer=red). Sorted by val_loss ascending.

   b. per_component_rmse.png: Grouped bar chart showing per-component RMSE in physical units for the top 5 models. Components: pos_x(mm), pos_y(mm), vel_x(mm/s), vel_y(mm/s), yaw(rad), omega_z(rad/s).

   c. error_histograms.png: 6-panel figure (2x3 grid) showing error distribution histograms for the best model, one panel per component (pos_x, pos_y, vel_x, vel_y, yaw, omega_z). Title shows RMSE value.

   d. predicted_vs_actual.png: 6-panel scatter plot (2x3 grid) for the best model showing predicted delta vs actual delta per component. Include diagonal reference line and R-squared annotation.

6. Prints a ranked summary table to stdout showing: rank, config name, arch type, val_loss, per-component RMSE.

7. CLI: `python3 -m aprx_model_elastica.analyze_sweep --output-base output/surrogate --figures-dir figures/surrogate_training`

Use matplotlib for all plots. Set figure DPI to 150.
  </action>

  </verify>
  <done>4 diagnostic plots saved to figures/surrogate_training/, per-component RMSE printed for all configs, ranked summary table printed</done>
