**\subsection{Resistive Force Theory Contact Model}** (~0.4 pages):
Write 2-3 sentences explaining that ground contact forces are modeled using Resistive Force Theory (RFT) — a slender-body approximation where the contact force at each rod element depends only on the local velocity components. Cite \citet{Till2019} or an appropriate RFT reference if available in knowledge/friction-contact-models.md.

Include the tangential and normal force equations: $f_t = -c_t v_t$ and $f_n = -c_n v_n$ where $c_t, c_n$ are anisotropic drag coefficients. State the values used in this work: $c_t = 0.01$, $c_n = 0.05$. Explain that anisotropy ($c_n > c_t$) is the physical mechanism enabling propulsion from a serpentine wave.

**\subsection{Central Pattern Generator Actuation}** (~0.4 pages):
Write 2-3 sentences explaining CPG actuation: the controller generates a target curvature wave $\kappa^\text{target}$ over the rod body; a proportional feedback law generates muscle moments to track this target.

Write the CPG equation: $\kappa_i^\text{target} = A \sin(k \cdot s_i - \omega t + \phi_0) + b$ where $s_i$ is the arc-length position of element $i$, $A$ is amplitude, $k$ wave number, $\omega = 2\pi f$ angular frequency, $\phi_0$ initial phase offset, and $b$ a constant turn bias. The 5-dimensional action vector $\mathbf{a} = (A, f, k, \phi_0, b)$ parameterizes this wave. State the physical ranges used (from knowledge/surrogate-mathematical-formulation.md).

**\subsection{Neural Surrogates for Soft Robots}** (~0.7 pages):
Write a cohesive narrative (not a bullet list) covering the literature on neural network surrogates for elastic rod / soft robot dynamics:

1. **DD-PINN** \citet{Stolzle2025}: Open with the strongest result — a domain-decoupled PINN achieving 44,000x speedup over numerical integration with <3mm end-effector error for full-state (72D) Cosserat rod dynamics. Explain the key innovation: decoupling the time domain to enable closed-form gradients.

2. **KNODE-Cosserat** \citet{Hsieh2024}: Hybrid approach combining a simplified physics model with a neural ODE residual correction. Achieves 58.7% accuracy improvement over the physics-only model on real hardware. Note that training data was collected from PyElastica — directly analogous to our setup.

3. **SoRoLEX** \citet{SoRoLEX2024}: LSTM forward dynamics model trained on soft robot simulator data, used as a surrogate environment for GPU-parallelized PPO (JAX). Most directly relevant to this work — the pipeline (sim data → neural env → RL) is the same approach.

4. **PINN for articulated soft robots** \citet{PINNSoftRobot2025}: 467x speedup, MPC at 47 Hz, demonstrating the approach scales to articulated systems with few training datasets.

Conclude the subsection by positioning the current work: we follow the data-driven approach of SoRoLEX but with a MLP (not LSTM) predicting full Cosserat rod state (124D), trained with both single-step MSE and multi-step rollout loss to reduce autoregressive drift.

**\subsection{Reinforcement Learning with Learned Simulators}** (~0.5 pages):
Write a cohesive narrative covering model-based RL and the use of learned world models:

1. **MBPO** \citet{Janner2019}: The canonical result showing that model-based policy optimization with learned ensemble dynamics models achieves model-free asymptotic performance with ~5x less real-env data.

2. **DreamerV3** (cite from knowledge/neural-surrogates-cosserat-rod-dynamics.md if available, else omit): Recurrent world model enabling RL in latent space — demonstrates that surrogate quality is a key bottleneck for downstream RL.

3. Connect to this work: The accuracy of the neural surrogate directly determines RL policy quality. Our comparison with a direct Elastica RL baseline (Phase 8) quantifies the policy gap attributable to surrogate modeling error.

Also mention GNN-based simulators \citep{SanchezGonzalez2020} and MeshGraphNets \citep{Pfaff2021} as graph-structured alternatives — note that for the small 20-node rod, an MLP suffices and the added complexity of GNNs is not warranted.

Use proper academic writing style: connect ideas with logical connectives ("Building on this..."; "In contrast to..."; "This motivates..."). Avoid bullet lists in this section. Use \citet{} for textual references and \citep{} for parenthetical.
