**src/pinn/nondim.py** -- Physics-based nondimensionalization (NOT z-score):
- Class `NondimScales` with constructor taking rod parameters (defaults from config):
  - `L_ref = snake_length` (default 1.0 m)
  - `t_ref = dt_ctrl` (default 0.5 s)
  - `V_ref = L_ref / t_ref` (2.0 m/s)
  - `omega_ref = 1.0 / t_ref` (2.0 rad/s)
  - `F_ref = E * I / L_ref**2` where `I = pi * r**4 / 4`, E=2e6, r=0.001 (characteristic elastic force)
  - Computed: `I = math.pi * 0.001**4 / 4 = 7.854e-13`, `F_ref = 2e6 * 7.854e-13 / 1.0**2 = 1.5708e-6`
- Methods `nondim_state(state_124)` and `redim_state(nondim_state)`:
  - pos_x, pos_y: divide/multiply by L_ref
  - vel_x, vel_y: divide/multiply by V_ref
  - yaw: already dimensionless (radians), no scaling
  - omega_z: divide/multiply by omega_ref
- Methods `nondim_delta(delta_124)` and `redim_delta(nondim_delta)` using same scales
- All operations work on (..., 124) tensors using the named slices from state.py

**src/pinn/regularizer.py** -- Physics regularizer with 4 constraint types:
- Class `PhysicsRegularizer(nn.Module)` with `__init__(self, dt: float = 0.5)`:
  - Stores dt as buffer
- Method `forward(self, state: Tensor, delta_pred: Tensor) -> Tensor`:
  - Operates on RAW 124-dim state (not relative 130-dim) because physics constraints are in absolute coordinates
  - Computes `next_state = state + delta_pred`
  - **Constraint 1 -- Kinematic consistency (x, y):**
    `loss_kin_x = MSE(delta_pred[..., POS_X], (state[..., VEL_X] + 0.5 * delta_pred[..., VEL_X]) * dt)`
    `loss_kin_y = MSE(delta_pred[..., POS_Y], (state[..., VEL_Y] + 0.5 * delta_pred[..., VEL_Y]) * dt)`
    `loss_kin = loss_kin_x + loss_kin_y`
  - **Constraint 2 -- Angular kinematic consistency:**
    `loss_ang = MSE(delta_pred[..., YAW], (state[..., OMEGA_Z] + 0.5 * delta_pred[..., OMEGA_Z]) * dt)`
  - **Constraint 3 -- Curvature-moment consistency (approximate):**
    Uses the constitutive law: `omega_dot_approx = B * (kappa - kappa_rest) / (rho * I * dl)`
    Since we don't have kappa directly from state, approximate from yaw differences:
    `kappa_approx = diff(state[..., YAW], dim=-1) / dl` where `dl = L / n_elements = 1.0/20 = 0.05`
    `kappa_next = diff(next_state[..., YAW], dim=-1) / dl`
    `omega_z_interior = state[..., OMEGA_Z[:-1]] + state[..., OMEGA_Z[1:]]` (avg at element boundaries)
    This is APPROXIMATE -- use low weight (0.1x the kinematic losses)
    `loss_curv = 0.1 * MSE(delta_pred[..., OMEGA_Z], omega_dot_approx * dt)` -- only for interior elements
    NOTE: This is the weakest constraint. If it causes instability, it can be disabled.
  - **Constraint 4 -- Energy conservation (approximate):**
    `KE = 0.5 * sum(vel_x^2 + vel_y^2)` (proportional, not exact mass-weighted)
    `KE_next = 0.5 * sum(next_vel_x^2 + next_vel_y^2)`
    `delta_KE = KE_next - KE`
    Energy penalty: `loss_energy = relu(abs(delta_KE) - energy_threshold)^2` where threshold is generous (e.g., 10.0 in nondim units)
    This prevents the model from predicting physically unreasonable energy changes.
  - Returns: `loss_kin + loss_ang + loss_curv + loss_energy`
  - All losses computed with F.mse_loss(reduction='mean')

**src/pinn/loss_balancing.py** -- ReLoBRaLo implementation:
- Class `ReLoBRaLo`:
  - `__init__(self, n_losses: int, alpha: float = 0.999, temperature: float = 1.0, rho: float = 0.999)`
  - `self.prev_losses: Optional[torch.Tensor] = None`
  - `self.weights = torch.ones(n_losses)` (initial weights all 1.0)
  - `self.lookback_losses: Optional[torch.Tensor] = None` (for random lookback)
  - Method `update(self, losses: List[torch.Tensor]) -> torch.Tensor`:
    - `current = torch.tensor([l.item() for l in losses])`
    - If `self.prev_losses is None`: store and return ones
    - Random lookback: with probability (1-rho), use lookback_losses instead of prev_losses
    - `ratios = current / (ref_losses + 1e-8)`
    - `new_weights = softmax(ratios / temperature) * n_losses`
    - `self.weights = alpha * self.weights + (1 - alpha) * new_weights`
    - Update prev_losses and lookback_losses
    - Return `self.weights`
