**src/pinn/physics_residual.py** -- Differentiable Cosserat rod RHS:

Class `CosseratRHS(nn.Module)`:
- `__init__(self, E=2e6, rho=1200.0, r=0.001, L=1.0, n_elem=20, ct=0.01, cn=0.1, eps=1e-6)`:
  - Compute and register as buffers: `I = pi*r^4/4`, `A = pi*r^2`, `B = E*I`, `S = E*A`, `dl = L/n_elem`, `mass = rho*A*dl`
  - Store friction coefficients ct, cn, eps

- Method `forward(self, state: Tensor) -> Tensor`:
  - Input: (B, 124) raw state tensor
  - Extract: pos_x (B,21), pos_y (B,21), vel_x (B,21), vel_y (B,21), yaw (B,20), omega_z (B,20)
  - **Kinematic equations (dx/dt):**
    - `d_pos_x/dt = vel_x` (21 nodes)
    - `d_pos_y/dt = vel_y` (21 nodes)
    - `d_yaw/dt = omega_z` (20 elements)
  - **Curvature from yaw:**
    - `kappa = diff(yaw, dim=-1) / dl` -- (B, 19) curvature at internal joints
  - **Internal elastic moment (bending):**
    - `moment = B * kappa` -- (B, 19) at joints
    - Torque on elements from bending: `tau_elastic[i] = (moment[i] - moment[i-1]) / dl` for interior, boundary elements get single-sided
  - **RFT friction forces (full anisotropic, differentiable):**
    - Compute tangent directions from positions: `tangent = diff(pos, dim=-1)`, normalize with `tangent / (norm + eps)`
    - Interpolate tangent to nodes (average neighbors, first/last use nearest element)
    - Tangential velocity: `v_tan = dot(v, tangent) * tangent`
    - Normal velocity: `v_norm = v - v_tan`
    - `F_friction = -ct * v_tan - cn * v_norm` (B, 2, 21)
    - Regularize tangent computation: use `tangent / sqrt(norm^2 + eps^2)` instead of `tangent / (norm + eps)` for gradient stability
  - **Force balance (Newton's 2nd law):**
    - `d_vel_x/dt = F_total_x / mass_node` (21 nodes)
    - `d_vel_y/dt = F_total_y / mass_node` (21 nodes)
    - F_total includes: friction + internal elastic forces projected to x,y
    - Internal elastic forces from bending: convert moment gradient to transverse force
  - **Angular momentum balance:**
    - `d_omega_z/dt = tau_total / (rho * I * dl)` (20 elements)
    - tau_total includes: elastic moments + any external torques
  - Returns: (B, 124) time derivative tensor [d_pos_x, d_pos_y, d_vel_x, d_vel_y, d_yaw, d_omega_z]

  NOTE: This is an APPROXIMATE f_SSM. The full Cosserat rod equations include shear, extension, and 3D effects. For a 2D snake on a plane, bending + friction + kinematic constraints capture the dominant dynamics. Stretching is omitted (inextensible rod approximation). The implementation must be validated against PyElastica outputs.

**src/pinn/collocation.py** -- Collocation point sampling:

Function `sample_collocation(n_points: int, t_start: float = 0.0, t_end: float = 0.5, method: str = "sobol") -> torch.Tensor`:
- Uses `scipy.stats.qmc.Sobol(d=1, scramble=True)` for Sobol quasi-random
- Returns sorted `(n_points,)` float32 tensor in [t_start, t_end]
- Also supports method="uniform" for comparison

Function `adaptive_refinement(residuals: torch.Tensor, t_colloc: torch.Tensor, n_new: int = 100, beta: float = 2.0) -> torch.Tensor`:
- RAR: probability of sampling near point i proportional to |residual_i|^beta
- Sample n_new new points from this distribution (with noise for diversity)
- Return (n_new,) tensor of new collocation points
