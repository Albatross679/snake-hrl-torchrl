**src/pinn/ansatz.py** -- Damped sinusoidal ansatz:

Class `DampedSinusoidalAnsatz(nn.Module)`:
- `__init__(self, state_dim: int = 130, n_basis: int = 5)`:
  - `self.state_dim = state_dim`
  - `self.n_basis = n_basis`
  - `self.param_dim = 4 * state_dim * n_basis` (alpha, delta_damp, beta, gamma)
  - Note: using `delta_damp` to avoid confusion with state delta

- `forward(self, params: Tensor, t: Tensor) -> Tensor`:
  - `params`: (B, 4 * m * n_g) from neural network
  - `t`: (N_c,) collocation times OR scalar for single evaluation
  - Parse params into (B, m, n_g) for each of alpha, delta_damp, beta, gamma
  - `delta_damp_pos = F.softplus(delta_damp)` -- ensure positive damping
  - If t is scalar, reshape to (1,)
  - `t_exp = t[None, :, None, None]` -- (1, N_c, 1, 1)
  - `phase = beta[:, None, :, :] * t_exp + gamma[:, None, :, :]` -- (B, N_c, m, n_g)
  - `damping = torch.exp(-delta_damp_pos[:, None, :, :] * t_exp)` -- (B, N_c, m, n_g)
  - `g = (alpha[:, None, :, :] * damping * (torch.sin(phase) - torch.sin(gamma[:, None, :, :]))).sum(-1)` -- (B, N_c, m)
  - Note: `sin(phase) - sin(gamma)` at t=0 gives `sin(gamma) - sin(gamma) = 0` -- EXACT IC
  - Returns: (B, N_c, m) or (B, m) if t was scalar

- `time_derivative(self, params: Tensor, t: Tensor) -> Tensor`:
  - Closed-form g_dot = d/dt [alpha * exp(-delta*t) * (sin(beta*t + gamma) - sin(gamma))]
  - `= alpha * exp(-delta*t) * [beta * cos(beta*t + gamma) - delta * (sin(beta*t + gamma) - sin(gamma))]`
  - Same shape as forward output
  - NO autodiff needed -- this is the key DD-PINN advantage

**src/pinn/models.py** -- DDPINNModel and FourierFeatureEmbedding:

Class `FourierFeatureEmbedding(nn.Module)`:
- `__init__(self, input_dim: int, n_features: int = 256, sigma: float = 10.0)`:
  - `self.B = nn.Parameter(torch.randn(input_dim, n_features) * sigma, requires_grad=False)`
- `forward(self, x: Tensor) -> Tensor`:
  - `proj = 2 * pi * x @ self.B`
  - Return `cat([sin(proj), cos(proj)], dim=-1)` -- output dim = 2 * n_features

Class `DDPINNModel(nn.Module)`:
- `__init__(self, state_dim=130, action_dim=5, time_encoding_dim=4, n_basis=5, hidden_dim=512, n_layers=4, n_fourier=128, fourier_sigma=10.0, dt=0.5)`:
  - `self.ansatz = DampedSinusoidalAnsatz(state_dim, n_basis)`
  - `self.dt = dt`
  - Input dim for NN = state_dim + action_dim + time_encoding_dim (same 139 as existing surrogates)
  - But wrap with Fourier features: `self.fourier = FourierFeatureEmbedding(139, n_fourier, fourier_sigma)`
  - NN: MLP with residual connections
    - `input_proj = Linear(2 * n_fourier, hidden_dim)`
    - `layers = [Linear(hidden_dim, hidden_dim) for _ in range(n_layers)]`
    - `output = Linear(hidden_dim, ansatz.param_dim)`
    - Zero-init output layer (initial prediction = zero params = zero delta)
  - `self.config = None` -- placeholder for SurrogateModelConfig compatibility

- `forward(self, state: Tensor, action: Tensor, time_encoding: Tensor) -> Tensor`:
  - **Same signature as SurrogateModel.forward()**
  - `x = cat([state, action, time_encoding], dim=-1)` -- (B, 139)
  - `h = tanh(input_proj(fourier(x)))` -- (B, hidden_dim)
  - For each layer: `h = h + tanh(layer(h))` -- residual
  - `params = output(h)` -- (B, ansatz.param_dim)
  - `delta = ansatz(params, torch.tensor([self.dt], device=x.device))` -- evaluate at t=dt
  - `delta = delta.squeeze(1)` -- (B, state_dim) -- single time point
  - Return delta

- `forward_trajectory(self, state: Tensor, action: Tensor, time_encoding: Tensor, t_colloc: Tensor) -> Tuple[Tensor, Tensor]`:
  - Same as forward but evaluates at multiple time points
  - Returns `(g, g_dot)` each of shape (B, N_c, state_dim)
  - Used during DD-PINN training for physics residual at collocation points

- `predict_next_state(self, state, action, time_encoding, normalizer=None) -> Tensor`:
  - Same pattern as SurrogateModel.predict_next_state()
  - `state_norm = normalizer.normalize_state(state) if normalizer else state`
  - `delta_norm = self.forward(state_norm, action, time_encoding)`
  - `delta = normalizer.denormalize_delta(delta_norm) if normalizer else delta_norm`
  - Return `state + delta`
