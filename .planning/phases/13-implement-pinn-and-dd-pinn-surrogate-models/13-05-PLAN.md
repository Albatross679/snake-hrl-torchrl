Create `src/pinn/train_pinn.py` as standalone DD-PINN training script:

**Entry point:** `python -m src.pinn.train_pinn`

**CLI args:**
- `--data-dir` (default: "data/surrogate_rl_step")
- `--n-basis` (default: 5, int) -- number of ansatz basis functions
- `--hidden-dim` (default: 512, int)
- `--n-layers` (default: 4, int)
- `--n-fourier` (default: 128, int) -- Fourier feature count
- `--fourier-sigma` (default: 10.0, float)
- `--n-collocation` (default: 100000, int) -- initial collocation points
- `--collocation-method` (default: "sobol")
- `--lambda-phys` (default: 1.0, float) -- physics loss weight
- `--epochs` (default: 300, int)
- `--lr` (default: 1e-3, float)
- `--use-lbfgs` (flag) -- enable L-BFGS refinement phase after Adam
- `--lbfgs-epochs` (default: 50, int)
- `--use-relobralo` (flag, default True)
- `--use-rar` (flag, default True) -- residual-based adaptive refinement
- `--rar-interval` (default: 20, int) -- refresh collocation every N epochs
- `--rar-fraction` (default: 0.1, float) -- fraction of points to replace
- `--curriculum-warmup` (default: 0.15, float)
- `--no-wandb` (flag)
- `--save-dir` (default: "output/surrogate/ddpinn")
- `--device` (default: "auto")
- `--patience` (default: 50)
- `--sweep` (flag) -- sweep n_basis in {5, 7, 10}

**Training loop:**
1. Load data with FlatStepDataset, convert to relative, fit StateNormalizer (same as train_regularized.py)
2. Create DDPINNModel(state_dim=130, n_basis=args.n_basis, ...)
3. Create CosseratRHS() for physics residual
4. Create NondimScales() for residual scaling
5. Create ReLoBRaLo(n_losses=2) if enabled
6. Generate initial collocation points: `t_colloc = sample_collocation(args.n_collocation, 0, 0.5, args.collocation_method)`

**Per-epoch training step:**
a. **Data loss (same mini-batch as regularized training):**
   - `delta_pred = model(state_norm, action, time_enc)` -- (B, 130)
   - `loss_data = F.mse_loss(delta_pred, delta_norm_target)`

b. **Physics loss at collocation points:**
   - Sample a mini-batch of training states: (B_phys, 130)
   - Keep raw states alongside for physics: (B_phys, 124)
   - `g, g_dot = model.forward_trajectory(state_norm, action, time_enc, t_colloc_batch)`
     - g: (B_phys, N_c_batch, 130) state deviations at collocation times
     - g_dot: (B_phys, N_c_batch, 130) closed-form time derivatives
   - Convert g to raw states: `state_at_t = relative_to_raw(raw_to_relative(raw_state)[:, None, :] + denorm(g))` -- (B_phys, N_c_batch, 124)
     - More precisely: the ansatz predicts deviation in normalized relative space. Denormalize, add to raw_to_relative(raw_state), then relative_to_raw to get raw states at collocation times.
   - `f_predicted = g_dot_raw` -- closed-form derivative in raw state space
   - `f_physics = rhs(state_at_t.view(-1, 124)).view(B_phys, N_c_batch, 124)` -- physics RHS
   - Nondimensionalize both: `scales.nondim_derivative(f_predicted)`, `scales.nondim_derivative(f_physics)`
   - `loss_phys = F.mse_loss(f_predicted_nondim, f_physics_nondim)`

c. **Curriculum + ReLoBRaLo:**
   - `phys_weight = curriculum(epoch, total_epochs, lambda_phys, warmup_frac)`
   - If ReLoBRaLo: `weights = balancer.update([loss_data, phys_weight * loss_phys])`

d. **RAR (every rar_interval epochs):**
   - Evaluate residual at all collocation points
   - Replace bottom rar_fraction with new points from adaptive_refinement()

e. **Adam + optional L-BFGS:**
   - Adam for `epochs` with cosine LR
   - If `--use-lbfgs`: switch to L-BFGS for `lbfgs_epochs` with lower lr (1e-4)
   - L-BFGS uses `torch.optim.LBFGS(max_iter=20, history_size=50)`

**W&B logging (project: snake-hrl-pinn):**
- loss_data, loss_phys, phys_weight, total_loss per epoch
- relobralo weights
- n_collocation (changes with RAR)
- val_loss, val_per_component_rmse

**Post-training evaluation:**
- Per-component RMSE on validation set (same as regularized training)
- Save eval_metrics.json to save_dir
- If --sweep: run n_basis in {5, 7, 10}, save to {save_dir}/n_basis_{N}/

**Checkpoint saving:**
- model.pt, normalizer.pt, config.json (same format as existing surrogates)
- Additional: ansatz_config.json with n_basis, hidden_dim, etc.

**Post-sweep plots:**
- `figures/pinn/ddpinn_training.png`: training curves for all n_basis values
- `figures/pinn/ddpinn_per_component.png`: per-component RMSE comparison (baseline + regularizer best + DD-PINN variants)
