    **4 Generic Probe PDEs** (not project-specific, per D-01):

    1. `ProbePDE1` -- 1D heat equation: u_t = alpha * u_xx
       - Analytical: u(x,t) = exp(-alpha * pi^2 * t) * sin(pi * x)
       - IC: u(x,0) = sin(pi*x), BC: u(0,t)=u(1,t)=0
       - Tests: data fitting + optimizer works
       - Pass: MSE vs analytical < 1e-4 in 500 epochs
       - `compute_loss(model)`: trains small MLP on (x,t)->u with MSE vs analytical at 1000 Sobol points + PDE residual at 500 collocation points

    2. `ProbePDE2` -- 1D advection: u_t + c * u_x = 0, c=1.0
       - Analytical: u(x,t) = sin(2*pi*(x - c*t))
       - Tests: BC/IC enforcement + PDE residual computation
       - Pass: PDE residual < 1e-3, BC error < 1e-4

    3. `ProbePDE3` -- 1D Burgers: u_t + u * u_x = nu * u_xx
       - Analytical: u(x,t) = -tanh((x - 0.5*t)/(2*nu)) for traveling wave, nu=0.01/(2*pi)
       - Tests: nonlinear PDE + loss balancing
       - Pass: both data and physics loss decrease, ratio stays < 100:1

    4. `ProbePDE4` -- 1D reaction-diffusion: u_t = D * u_xx + k * u * (1-u), D=0.01, k=1.0
       - Analytical: traveling wave u(x,t) = 1/(1 + exp(-(x - c*t)/sqrt(6*D/k))) with c = 5*sqrt(D*k/6)
       - Tests: multi-scale + requires Fourier features to capture sharp front
       - Pass: captures both low and high-freq modes (FFT of prediction has energy at correct frequencies)

    Each probe class has:
    - `name: str` -- e.g., "heat_1d"
    - `max_epochs: int` -- training budget (500 for simple, 1000 for complex)
    - `model_kwargs: dict` -- kwargs for _ProbeMLP (hidden_dim=64, n_layers=2)
    - `analytical_solution(x, t, **params) -> Tensor`
    - `pde_residual(u, grads_dict) -> Tensor` -- computes PDE residual given solution and its derivatives
    - `initial_condition(x) -> Tensor`
    - `boundary_conditions() -> dict` -- maps "x=0", "x=1" etc to values
    - `compute_loss(model) -> Tensor` -- full loss (data + physics) for one training step
    - `check_pass(model) -> bool` -- evaluates pass criterion
    - `pass_criterion(metrics_dict) -> bool` -- checks metrics against thresholds
