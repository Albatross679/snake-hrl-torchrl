    ```python
    class PINNDiagnostics:
        """Non-invasive diagnostic middleware for PINN training.

        Mirrors src/trainers/diagnostics.py pattern for RL.
        Computes and logs PINN-specific metrics each epoch.
        """

        def __init__(self, wandb_run=None, ntk_interval: int = 50, n_params_sample: int = 500):
            self.wandb_run = wandb_run
            self.ntk_interval = ntk_interval  # per D-09
            self.n_params_sample = n_params_sample  # per D-10
            self._history = {
                "loss_ratio": deque(maxlen=100),
                "grad_norm_data": deque(maxlen=100),
                "grad_norm_phys": deque(maxlen=100),
                "residual_mean": deque(maxlen=100),
            }
    ```

    **Compute methods** (per D-08):

    1. `compute_loss_ratio(self, loss_data: Tensor, loss_phys: Tensor) -> float`:
       - Returns loss_phys.item() / max(loss_data.item(), 1e-10)
       - Appends to self._history["loss_ratio"]

    2. `compute_per_loss_gradients(self, model: nn.Module, loss_data: Tensor, loss_phys: Tensor) -> dict`:
       - Computes separate gradient norms for each loss term using backward(retain_graph=True)
       - Returns {"diagnostics/grad_norm_data": float, "diagnostics/grad_norm_phys": float, "diagnostics/grad_norm_ratio": float}
       - Restores model.zero_grad() after computation
       - IMPORTANT: Must call model.zero_grad() before each backward, and restore original gradients after

    3. `compute_residual_statistics(self, residuals: Tensor) -> dict`:
       - residuals: (N, state_dim) or (N,) tensor
       - Returns {"diagnostics/residual_mean": float, "diagnostics/residual_max": float, "diagnostics/residual_std": float, "diagnostics/residual_p95": float}

    4. `compute_relobralo_health(self, weights: Tensor) -> dict`:
       - weights: (n_losses,) tensor from ReLoBRaLo.weights
       - Returns {"diagnostics/relobralo_w_data": float, "diagnostics/relobralo_w_phys": float, "diagnostics/relobralo_ratio": float}

    5. `compute_per_component_violations(self, residuals: Tensor) -> dict`:
       - residuals: (N, 124) tensor of per-state-component physics violations
       - Returns per-component group means: {"diagnostics/violation_pos_x": float, "diagnostics/violation_vel_x": float, "diagnostics/violation_yaw": float, "diagnostics/violation_omega_z": float}
       - Uses slices from src/pinn/_state_slices.py: POS_X, VEL_X, YAW, OMEGA_Z
       - Read _state_slices.py first to verify the exact constant names before importing

    **NTK computation** (per D-09, D-10, D-11):

    `compute_ntk_eigenvalues(model: nn.Module, sample_inputs: Tensor, n_params_sample: int = 500) -> dict`:
    - Standalone function (not method) for reuse
    - sample_inputs: (N, input_dim) tensor -- a small batch of collocation inputs
    - Subsamples n_params_sample parameters from model (per D-10)
    - Computes Jacobian J of model output w.r.t. subsampled parameters using torch.autograd.functional.jacobian or manual per-sample gradient accumulation
    - Computes NTK approximation: K = J @ J.T
    - Returns {"ntk/eigenvalue_max": float, "ntk/eigenvalue_min": float, "ntk/condition_number": float, "ntk/spectral_decay_rate": float} (per D-11)
    - Handles edge cases: if model has fewer params than n_params_sample, use all params

    **log_step method:**
    ```python
    def log_step(self, epoch, model, loss_data, loss_phys, balancer=None, residuals=None, collocation_inputs=None):
        """Compute all diagnostics for one epoch and return merged dict.

        Called once per epoch from train_pinn.py. Per D-06: non-invasive.
        Per D-07: logs to W&B only (no alerts, no auto-stopping).
        """
        metrics = {}

        # Always compute
        metrics["diagnostics/loss_ratio"] = self.compute_loss_ratio(loss_data, loss_phys)

        # ReLoBRaLo health (if balancer provided)
        if balancer is not None:
            metrics.update(self.compute_relobralo_health(balancer.weights))

        # Residual stats (if residuals provided)
        if residuals is not None:
            metrics.update(self.compute_residual_statistics(residuals))
            if residuals.dim() == 2 and residuals.shape[1] >= 124:
                metrics.update(self.compute_per_component_violations(residuals))

        # NTK (expensive, per D-09: every ntk_interval epochs)
        if collocation_inputs is not None and epoch % self.ntk_interval == 0:
            ntk_metrics = compute_ntk_eigenvalues(model, collocation_inputs, self.n_params_sample)
            metrics.update(ntk_metrics)

        # Log to W&B (per D-07: log-only, no alerts)
        if self.wandb_run is not None:
            try:
                import wandb
                wandb.log(metrics, step=epoch)
            except Exception:
                pass

        return metrics
    ```
