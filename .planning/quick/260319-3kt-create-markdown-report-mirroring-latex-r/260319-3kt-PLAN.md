**Document structure:**

- `\section{X}` -> `# X`, `\subsection{X}` -> `## X`, `\subsubsection{X}` -> `### X`
- Table of contents: not needed (Markdown renderers auto-generate)

- `\renewcommand{\arraystretch}` -> omit (Markdown handles spacing)
- `\centering` -> omit
- Table captions (`\caption{X}`) -> italic text below: `*Table N: X*`
