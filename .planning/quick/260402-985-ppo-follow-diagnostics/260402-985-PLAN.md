**Probe subcommand** (non-training diagnostics, less than 5 min total):

EXP1 - Reward Landscape Probe:
- Create single SoftManipulatorEnv with default follow_target config
- Run 50 episodes with random actions (sampled uniform [-1, 1])
- Collect per-step: reward, dist_to_goal, tip_pos, target_pos
- Compute and report: mean/std/min/max/median reward, mean/std distance,
  reward histogram (bins: [0, 0.001, 0.01, 0.05, 0.1, 0.5, 1.0]),
  fraction of steps with reward > 0.01, > 0.05, > 0.1
- Also compute exp(-2*d) and exp(-1*d) on same distances for comparison
- Save results JSON to output/diagnostics/exp1_reward_landscape.json

EXP2 - Action Impact Probe:
- Run 10 episodes with zero actions (all zeros) and 10 with random actions
- Per step, measure: tip displacement (||tip_t - tip_{t-1}||), total state change (||obs_t - obs_{t-1}||)
- Report: mean/std tip displacement per step for zero vs random,
  mean total state change for zero vs random,
  ratio (random_displacement / zero_displacement) -- if ratio near 1.0, actions have no effect
- Save to output/diagnostics/exp2_action_impact.json

EXP3 - Reachability Analysis:
- Sample 200 target positions using TargetGenerator (from choi2025.tasks)
- For efficiency: pick 20 targets, run 10 random-action rollouts of 100 steps each per target
- Track minimum distance achieved to each target across all rollouts
- Report: fraction of targets where min_dist < 0.3, < 0.1, < 0.05
- Break down by hemisphere: upper (z>0) vs lower (z<=0) vs equatorial (|z|<0.15)
- Save to output/diagnostics/exp3_reachability.json

**Background**: PPO shows no learning on follow_target (mean dist 0.884m, CoV 23.4%). SAC learns on same task. This experiment systematically tests 6 hypotheses.

**Hypotheses** (numbered H1-H6):
- H1 Reward Sparsity: exp(-5*d) at d=0.88 gives ~0.012. Near-zero gradient signal. Tested by EXP1 (probe) and EXP5 (dense reward training).
- H2 Integrative Action Space: Delta curvature accumulates, making credit assignment hard. Tested by EXP2 (action impact) and EXP4 (static target removes one dynamic).
- H3 Mock Physics: Simplified dynamics (damped by 0.1*dt) may not respond meaningfully. Tested by EXP2 (action impact probe).
- H4 3D Reachability: Upper hemisphere targets may be unreachable under gravity. Tested by EXP3 (reachability analysis).
- H5 Observation Overload: 148 dims overwhelm PPO's 8192-frame batches. Tested by EXP7 (reduced obs).
- H6 Network Too Large: 4x512 (~1M params) with small batches = insufficient gradient updates. Tested by EXP8 (small network).
